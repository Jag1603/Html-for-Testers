Windows Keyboard Shortcuts

Copy (Ctrl + C)
Copies the selected text or item to the clipboard.
Paste (Ctrl + V)
Pastes the copied or cut text or item from the clipboard.
Cut (Ctrl + X)
Removes the selected text or item and copies it to the clipboard.
Undo (Ctrl + Z)
Reverses the last action.
Redo (Ctrl + Y)
Reapplies the last undone action.
Save (Ctrl + S)
Saves the current document or file.
Find (Ctrl + F)
Opens the search box to find text within the current document.
Print (Ctrl + P)
Opens the print dialog to print the current document.
New Window (Ctrl + N)
Opens a new window in the current application.
Switch Applications (Alt + Tab)
Switches between open applications.
Lock Computer (Win + L)
Locks your computer screen.
Task Manager (Ctrl + Shift + Esc)
Opens the Task Manager to view and manage running processes.
Open File Explorer (Win + E)
Opens the File Explorer.

Notes

Some shortcuts may differ depending on the version of Windows or customizations made to the system.
